DevOps

Microclimate is an end-to-end development environment that lets you rapidly create, edit, and deploy cloud-native applications. Applications are run in containers from day one and can be delivered into production on Kubernetes through an automated DevOps pipeline using Jenkins. Microclimate can be installed locally or on IBM Cloud Private and currently supports Java, Node.js, Swift, Go, Python and generic Docker projects.

 

Microclimate is not just another friendly IDE! Microclimate includes application monitoring tools and an HTTP load-driver to start. Each of these tools is preinstalled and preconfigured for your service so you can switch tabs and go. You no longer need to find other tools or try to set them up when you run into problems.

Microclimate enables you to create your microservices and automated build in Docker immediately. No more time spent trying to re-create problems that happen on one machine, or differences when you are switching to containers in production. The environment is built by using Docker too. You can run it locally, or host it on your cloud platform and use it over the web. No more issues with local settings or installers for multiple tools.
