Hybrid Integration

IBM Cloud Automation Manager is a multi-cloud, self-service management platform running on IBM Cloud Private that allows you to expose cloud services to any cloud through the IBM Cloud Private catalog. It accelerates developer velocity and improve governance.

Cloud Automation Manager uses open source Terraform to manage and deliver cloud infrastructure as code. The Cloud Automation Manager content library comes pre-populated with sample templates to help you get started quickly.

A Chef runtime environment can also be deployed using Cloud Automation Manager for more advanced application configuration and deployment. The Cloud Automation Manager Template Designer accelerates development of reusable Terraform automation.
